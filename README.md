
# Relay with AC PS and STM32

This is my PCB Design practice project.
Flexible programming expandable with sensors through I2C.


## Features

- AC Power Supply to 5V, 3V3
- STM32F103C8T6
- SWD J Tag Connector
- USB Type C 2.0
- USART, I2C
- 5V Relay
